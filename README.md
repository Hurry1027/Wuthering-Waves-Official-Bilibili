## 项目介绍

本程序实现鸣潮Bilibili与官方客户端版本间的备份以及快捷转换，省去了转换时需要重新读盘下载修复游戏文件的操作。

### 原理
- 对鸣潮KrPcSdk_Mainland文件夹进行备份和恢复，实现鸣潮B服和官服之间的转换。
- 自动识别当前版本是B服还是官服。

### 应用场景
- 有B服和官服两个账号需要频繁切换的电脑端玩家。

### 1. 安装Python
Python是运行 switch.py 文件的基础，你可以从Python官方网站下载并安装Python。

- 步骤 ：
  1. 访问 [Python官方下载页面](https://www.python.org/) 。
  2. 根据你的操作系统（Windows、Mac OS、Linux）选择合适的Python版本（建议选择Python 3.7及以上版本）进行下载。
  3. 运行下载的安装程序，在安装过程中，确保勾选 “Add Python to PATH” 选项，这将允许你在命令行中直接使用Python。
  4. 按照安装向导的提示完成安装。
  5. 安装完成后，你可以通过在命令行中输入 python --version 来验证Python是否成功安装。
### 2. 下载官服和B服客户端
  1. 分别下载官服和B服客户端
  2. 打开设置，将游戏目录设为相同目录

### 3. 填写配置文件
- 步骤 ：
  1. 打开 config.json 文件。
  2. 根据你的实际情况，修改以下配置项：
      - “gameFolderPath”: 设置游戏目录(Wuthering Waves Game文件夹)。
      - ”client_launch_paths“: 分别设置B服客户端和官服客户端启动路径。
  3. 保存并关闭 config.json 文件。

### 4. 运行 鸣潮B服官服转换及启动工具.bat 或 运行 switch.py
- 你可以直接运行 鸣潮B服官服转换及启动工具.bat ，它会自动运行switch.py。
- switch.py运行步骤 ：
  1. 打开cmd或命令提示服。
  2. 使用 cd 命令切换到 switch.py 文件所在的目录。例如：
  3. 运行 switch.py 文件：
     ```bash
     python switch.py
      ```
### 5. 初始化流程（以当前版本为B服举例）
1. 备份Bilibili版本：运行鸣潮B服官服转换及启动工具.bat，选择备份文件，程序会进行当前版本文件的备份，然后选择取消转换。(即使选择转换也会显示：目标版本（Official）的备份不存在)。
2. 手动切换到Official版本： 打开官方客户端，点击修复文件，等待修复完成。
3. 备份Official版本：运行鸣潮B服官服转换及启动工具.bat，选择备份文件，程序会进行当前版本文件的备份，此时已完成了初始化，你可以选择继续转换，或者取消转换后退出程序。

### 6. 官B服转换流程
1. 运行鸣潮B服官服转换及启动工具.bat，选择备份文件并转换，程序会自动备份当前KrPcSdk_Mainland文件夹，并从目标版本的备份文件恢复该文件夹，从而转换到对应版本。
2. 转换完成后自动启动目标版本的客户端。


###  7. 大版本更新（如鸣潮2.4-2.5更新）
1. 正常预下载，最好在你主要玩的服进行预下载。
2. 版本更新后，切换到预下载时候的服，点击更新按钮。
3. 运行本程序，选择备份。
4. 手动切换到另一版本（打开另一版本客户端点击修复文件），等待扫描文件修复完成。
5. 运行本程序，选择备份。

**以上是最稳妥的做法，在第2步完成后，你也可以尝试跳过扫描修复步骤：**
在预下载的服更新完成后，当你需要转换到另一个服时，
可以使用[WinMerge](https://winmerge.org/downloads/?lang=zh_cn)软件对比KrPcSdk_Mainland文件夹差异，
如果差异较小，你可以手动修改备份中的差异文件。
**例如：** 2.7版本对比2.6版本，
官服KrPcSdk_Mainland文件夹仅有KRSDKConfig.json中的版本号(KR_GameVersion)不同，
这时如果你是在B服完成的版本更新，想要切换到官服，
在运行本程序前，你可以在Official文件夹（官服备份）下找到KRSDKConfig.json，手动将版本号(KR_GameVersion)修改为2.7.1，
然后启动本程序，选择备份并转换到Official版本。
此时就不需要再扫描修复，直接就可以官服启动成功。

### 8. 注意事项
- 初始化流程需认真按步骤进行，如果操作错误需要重来。
- 如果本程序出错导导致游戏无法启动，用客户端修复文件即可。
